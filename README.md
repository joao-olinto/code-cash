# CodeCash 💰

CodeCash is a simple banking system developed in Java to practice Object-Oriented Programming and backend architecture concepts.

## Features
- Create accounts
- Deposit money
- Withdraw money
- Remove accounts
- List accounts
- Custom exception handling
- Service layer architecture

## Technologies
- Java
- OOP (Inheritance, Polymorphism, Encapsulation)
- Streams API
- Collections
- Regex validation

## Structure
Program → Service → Domain (Entities)
